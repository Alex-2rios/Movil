# TerracotaWeb2p

Frontend web de maqueta para Terracota 2P construido con Flask + Python.

## Ejecutar

```bash
cd TerracotaWeb2p
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Abrir:

```txt
http://localhost:5000
```

## Interfaces

- Login
- Inicio / Estadísticas
- Gestión de usuarios
- Agregar usuario
- Editar usuario
- Reportes
- Gestión de inventario
- Agregar producto
- Editar producto

Todas las pantallas son cascarón navegable con datos de ejemplo.
